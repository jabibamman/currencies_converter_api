## Description

Convert your favorites currencies with this API :D.

You can find the documentation [here](https://github.com/jabibamman/currencies_converter_api/blob/main/docs/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
